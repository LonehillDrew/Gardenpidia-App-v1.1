Settings

Your key is stored only in this browser, in local storage, and is sent only to Anthropic when you make a request. It is never written to the repository or to GitHub. Do not enter your key on a shared or public computer. To revoke it, clear it here or rotate the key in your Anthropic console.

About this build

This site applies the GardenPedia house rules to every request: UK English, no em dashes, a South African default scope, metric units, Rands, and DD/MM/YYYY dates. It separates verified facts from assumptions, flags gaps as [MISSING INFO], and asks for named, verifiable sources.

Reconstructed rules. The behaviour rules were rebuilt from prior GardenPedia work, not read from your canonical skill repo. They all live in one file, assets/gardenpedia-config.js. When you push your real skill files, reconcile that file against them so the site and the skill stay in step.

Fact Finder format. Four fixed categories, ten facts each. Every fact is checked in the browser for character count (223 to 354), word count (35 to 59), space count (35 to 58), and three sentences. The metric strip under each card is the specimen label: green means within range, ochre means out of range. Machine validation is shown, but you remain the editor.

Connection. The site calls the Anthropic API directly from your browser using a key you provide in Settings. There is no server and no proxy, so nothing about your key touches GitHub. If you later want the key hidden from the browser entirely, the frontend can point at a small serverless proxy with almost no change.
